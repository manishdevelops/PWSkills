HTML v/s HTML5

HTML HTML5
HTML stands for HyperText Markup Language. HTML5 is the fifth version of HTML introduced in 2014.
HTML4 had limited support for multimedia and lacked native video and audio elements. HTML5 introduced native support for embedding audio and video using the <audio> and <video> elements.
HTML4 did not have specific semantic elements like <article>, <section>, and <nav>. HTML5 introduced semantic elements like <article>, <section>, <nav>, and more, providing better document structure.
