List of Tags.

  1. <html>: The root element that wraps the entire HTML document.
  2. <head>: Contains meta-information about the HTML document, such as title and character set.
  3. <title>: Sets the title of the HTML document, displayed in the browser's title bar or tab.
  4. <body>: Contains the content of the HTML document, including text, images, links, etc.
  5. <h1> to <h6>: Heading tags, used to define headings of various levels.
  6. <p>: Defines a paragraph of text.
  7. <a>: Creates hyperlinks, allowing navigation to other web pages.
  8. <img>: Embeds images in the HTML document.
  9. <ul>: Defines an unordered list, used with <li> for list items.
  10. <ol>: Defines an ordered list, used with <li> for list items.
  11. <li>: Represents a list item within <ul> or <ol>.
  12. <div>: A container element used for grouping and applying styles to content.
  13. <span>: An inline container used for applying styles to specific portions of text.
  14. <strong>: Represents strong importance, typically displayed as bold text.
  15. <em>: Represents emphasized text, typically displayed as italicized.
